==============================================================
  الخيار للسيارات — نظام الموارد البشرية v4
  دليل التثبيت والنشر
==============================================================

━━ للمستخدم النهائي (على كمبيوتر الشركة) ━━━━━━━━━━━━━━━━━━━

1. انقر نقراً مزدوجاً على:  AlkhayarHR_Setup_v4.0.exe
2. اضغط "التالي" → "التالي" → "تثبيت"
3. سيُنشأ اختصار على سطح المكتب تلقائياً
4. اضغط "إنهاء" → يفتح البرنامج مباشرة

لتشغيل البرنامج لاحقاً: انقر على أيقونة "الخيار HR" على سطح المكتب
أو من قائمة ابدأ → "الخيار HR"

لإلغاء التثبيت: لوحة التحكم → البرامج → "الخيار HR" → إلغاء التثبيت

━━ أين تُخزَّن البيانات؟ ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

قاعدة البيانات:
  C:\Users\<اسم المستخدم>\AppData\Roaming\AlkhayarHR\hr_data\

النسخ الاحتياطية:
  C:\Users\<اسم المستخدم>\AppData\Roaming\AlkhayarHR\backups\

⚠ مهم: البيانات محفوظة في AppData — لن تُحذف عند تحديث البرنامج
⚠ انسخ مجلد backups بانتظام إلى USB أو شبكة خارجية

━━ للمطوّر: بناء الـ .exe (مرة واحدة فقط) ━━━━━━━━━━━━━━━━━━

المتطلبات:
  ✔ Windows 10/11  (64-bit)
  ✔ Python 3.11+   — https://www.python.org/downloads/
      (اختر "Add Python to PATH" أثناء التثبيت)
  ✔ Inno Setup 6   — https://jrsoftware.org/isdl.php
      (لإنشاء ملف الإعداد الواحد Setup.exe)
  ✔ اتصال بالإنترنت للمرة الأولى فقط

خطوات البناء:
  1. افتح مجلد المشروع
  2. انقر نقراً مزدوجاً على  build_exe.bat
  3. انتظر 5-8 دقائق حتى ينتهي
  4. ستجد الناتج في:  installer_output\AlkhayarHR_Setup_v4.0.exe
  5. أرسل هذا الملف الواحد إلى الكمبيوتر الهدف

━━ هيكل ملفات البناء ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  launcher.py          — نقطة تشغيل البرنامج (Streamlit + WebView)
  alkhayar_hr.spec     — إعدادات PyInstaller
  installer.iss        — إعدادات Inno Setup (ملف الإعداد النهائي)
  build_exe.bat        — سكريبت البناء الكامل (شغّله وانتظر)

━━ استكشاف الأخطاء ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

• النافذة لا تفتح:   تأكد أن التثبيت اكتمل ونفّذ Setup.exe كـ Administrator
• شاشة بيضاء 5-10 ث: طبيعي، النظام يُهيّئ نفسه
• "المنفذ مشغول":    أغلق أي نسخة أخرى من البرنامج تعمل في الخلفية
